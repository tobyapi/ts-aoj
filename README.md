# ts-aoj

[Aizu Online Judge](https://onlinejudge.u-aizu.ac.jp/home) client for TypeScript.

# Usage
```ts
import * as aoj from 'ts-aoj';

aoj.user.get("tobias").then(user => console.log(user));
```
